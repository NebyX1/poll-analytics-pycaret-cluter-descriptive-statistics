Nota Importante!
El entorno virtual debe correr sobre Python 3.9 que es en el que fue probado con éxito, con Python 3.11 no funciona, con Python 3.10 funciona en
Anaconda Jupyter Notebooks.

Paso 1: Crear el entorno virutal en la carpeta del proyecto:
python -m virtualenv venv

Paso 2: Para activar el entorno virtual para empezar a trabajar, solo hay que escribir en la carpeta raíz:
venv\Scripts\activate

Paso 3: instala el contenido del archivo requirements.txt:
pip install -r requirements.txt

Paso Potencial: pycaret puede presentar un lote de problemas respecto a sus dependencias, solo aceptará en algunos casos hasta python 3.9
En caso de tener este problema, hay que bajar desde la web de python la versión 3.9 y luego crear el entorno virtual con una línea similar a esta:
"& 'C:\Users\Neby X\AppData\Local\Programs\Python\Python39\python.exe' -m venv --without-pip venv
.\venv\Scripts\Activate.ps1" en la consola de comandos
Naturalmente que hay que cambiar la dirección a donde se tenga esa versión de python instalada.
